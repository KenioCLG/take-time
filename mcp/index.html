Available Tools

Ferramentas Disponíveis

These are the MCP tools your AI assistant will have access to:

Estas são as ferramentas MCP que seu assistente de IA terá acesso:

list_blocks List study blocks with optional date range and status filters Listar blocos de estudo com filtros opcionais de data e status
create_block Create a new study block with subject, time, and topic Criar novo bloco de estudo com atividade, horário e tópico
update_block Update an existing block (time, subject, done status) Atualizar um bloco existente (horário, atividade, status)
delete_block Delete a study block by ID Excluir um bloco de estudo por ID
list_subjects List all activities/subjects with their profiles and slots Listar todas as atividades/matérias com perfis e horários
create_subject Create a new activity (study, training, or routine type) Criar nova atividade (tipo estudo, treino ou rotina)
get_stats Get completion stats, streaks, heatmap data, and weekly summary Obter estatísticas de conclusão, sequências, heatmap e resumo semanal
get_schedule Get the full weekly schedule with all time slots Obter a agenda semanal completa com todos os horários
suggest_schedule AI-powered schedule suggestions based on user patterns and goals Sugestões de horário baseadas em IA usando padrões e metas do usuário

Quick Setup

Instalação Rápida

  1. Install the server

    Install the Take Time MCP server globally via npm:

  2. Instale o servidor

    Instale o servidor MCP do Take Time globalmente via npm:

terminal # Install globally (optional)
npm install -g @taketimemcp/mcp-server

  1. Configure your AI client

    Add the server to your Claude Desktop, Cursor, or any MCP-compatible client:

  2. Configure seu cliente de IA

    Adicione o servidor no Claude Desktop, Cursor ou qualquer cliente compatível com MCP:

claude_desktop_config.json {
  "mcpServers": {
    "taketime": {
      "command": "npx",
      "args": ["@taketimemcp/mcp-server"],
      "env": {
        "TAKETIME_API_KEY": "your-api-key"
      }
    }
  }
}

  1. Get your API key

    Log in to Take Time, go to Settings, and generate your API key. Paste it in the config above.

  2. Obtenha sua chave de API

    Faça login no Take Time, vá em Configurações e gere sua chave de API. Cole no config acima.

  1. Start using!

    Ask your AI things like "Create a study block for JavaScript from 9am to 10am" or "How many blocks did I complete this week?"

  2. Comece a usar!

    Peça coisas como "Crie um bloco de estudo de JavaScript das 9h às 10h" ou "Quantos blocos concluí esta semana?"
