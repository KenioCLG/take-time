API Documentation

Complete reference for all 22 tools in the Take Time MCP Server.

Authentication

The MCP server supports three auth methods. Refresh token is recommended — copy it from the app's MCP settings panel.

MethodEnv VarAuto-refresh
Refresh TokenTAKETIME_REFRESH_TOKENYes (recommended)
Email/PasswordTAKETIME_EMAIL + TAKETIME_PASSWORDYes
Access TokenTAKETIME_API_KEYNo (~1h)

Schedule (Blocks)

read list_blocks

List study blocks with optional date range and status filters. Returns today's blocks by default.

Parameters

NameTypeDescription
datestringoptionalFilter by date (YYYY-MM-DD). Defaults to today.
date_fromstringoptionalStart of date range
date_tostringoptionalEnd of date range
subject_namestringoptionalFilter by activity name
donebooleanoptionalFilter by completion status

read get_schedule

Get the full weekly schedule with all blocks organized by day.

Parameters

NameTypeDescription
week_offsetnumberoptional0 = current week, -1 = last, 1 = next

write create_block

Create a new study block. Automatically checks for time conflicts.

Parameters

NameTypeDescription
subject_namestringrequiredActivity name
datestringrequiredDate (YYYY-MM-DD)
startstringrequiredStart time (HH:MM)
endstringrequiredEnd time (HH:MM)
topicstringoptionalTopic name

write update_block

Update an existing block — mark done, reschedule, set syllabus topic, toggle repeat.

Parameters

NameTypeDescription
block_idstringrequiredBlock ID
startstringoptionalNew start time (HH:MM)
endstringoptionalNew end time (HH:MM)
topicstringoptionalNew topic text
donebooleanoptionalMark as done/undone
selected_syllabus_idstringoptionalActive syllabus topic ID
repeat_dailybooleanoptionalToggle daily auto-repeat

delete delete_block

Permanently delete a study block.

Parameters

NameTypeDescription
block_idstringrequiredBlock ID to delete
