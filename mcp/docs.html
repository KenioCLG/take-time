API Documentation

Complete reference for the Take Time MCP Server tools.

Authentication

The MCP server authenticates via an API key passed as an environment variable. Each user generates their own key in the Take Time app under Settings.

// Environment variable TAKETIME_API_KEY="tt_live_abc123..."

read list_blocks

Retrieve study blocks with optional filters. Returns an array of block objects.

Parameters

NameTypeDescription
datestringoptionalFilter by date (YYYY-MM-DD)
date_fromstringoptionalStart of date range
date_tostringoptionalEnd of date range
subject_idstringoptionalFilter by subject/activity ID
donebooleanoptionalFilter by completion status

Response

[{ "id": "blk_abc123", "subject_id": "sub_xyz", "subject_name": "JavaScript", "date": "2026-06-11", "start": "09:00", "end": "10:00", "topic": "Ch. 3 - Arrays", "done": false }]

write create_block

Create a new study block on the user's schedule.

Parameters

NameTypeDescription
subject_idstringrequiredActivity/subject ID
datestringrequiredDate (YYYY-MM-DD)
startstringrequiredStart time (HH:MM)
endstringrequiredEnd time (HH:MM)
topicstringoptionalTopic or syllabus item

write update_block

Update an existing study block. Only pass the fields you want to change.

Parameters

NameTypeDescription
block_idstringrequiredBlock ID to update
subject_idstringoptionalNew activity ID
startstringoptionalNew start time
endstringoptionalNew end time
donebooleanoptionalMark as done/undone

delete delete_block

Permanently delete a study block.

Parameters

NameTypeDescription
block_idstringrequiredBlock ID to delete

read list_subjects

List all activities/subjects with their type, color, slots, and content (syllabus items, exercises, or habits).

Parameters

NameTypeDescription
typestringoptionalFilter by type: study, training, or inactive

Response

[{ "id": "sub_xyz", "name": "JavaScript", "type": "study", "color": "#6366f1", "slots": [{ "days": [1,3,5], "start": "09:00", "end": "10:00" }], "syllabus": [{ "title": "Arrays", "duration": 30, "done": false }] }]

write create_subject

Create a new activity/subject profile.

Parameters

NameTypeDescription
namestringrequiredActivity name (max 40 chars)
typestringrequiredstudy, training, or inactive
colorstringoptionalHex color (default: auto-assigned)
slotsarrayoptionalWeekly time slots

read get_stats

Get completion statistics, streaks, and heatmap data for the user.

Parameters

NameTypeDescription
periodstringoptionaltoday, week, month, or all (default: week)

Response

{ "total_blocks": 42, "completed_blocks": 35, "completion_rate": 0.83, "streak_days": 7, "total_minutes": 2100, "heatmap": { "2026-06-11": 5, "2026-06-10": 3 } }

read get_schedule

Get the user's full weekly schedule with all configured time slots per subject.

Parameters

NameTypeDescription
week_offsetnumberoptional0 = current week, -1 = last week, 1 = next week

write suggest_schedule

Get AI-powered schedule suggestions based on user's completion patterns, available time slots, and study goals.

Parameters

NameTypeDescription
goalstringoptionalNatural language goal (e.g. "study more math")
available_hoursnumberoptionalMax hours per day available
