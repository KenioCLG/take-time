## What is the Take Time CLI?

The Take Time CLI (`@taketime/cli`) gives you full control over your study schedule, activities, blocks, priorities, and settings — all from the terminal. It connects to the same Supabase database as the web app and MCP server.

```bash
taketime today
```

## How it works

```
┌─────────────────┐     HTTPS      ┌─────────────────┐
│   Take Time     │ ◄────────────► │  Take Time API  │
│   CLI           │    (REST)      │  (Supabase)     │
└─────────────────┘                └─────────────────┘
```

The CLI authenticates once with your email/password and persists a refresh token at `~/.taketime/`. All commands talk directly to the Supabase PostgREST API — the same relational tables used by the web app and MCP server.
