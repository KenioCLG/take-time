---
title: List Subjects
api: GET /subjects
description: "List all activities/subjects with their profiles, slots, and content."
---

<ParamField query="type" type="string">
  Filter by type: `study`, `training`, or `inactive`
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://api.taketime.space/v1/subjects" \
  -H "Authorization: Bearer tt_live_your_key_here"
```
</RequestExample>
