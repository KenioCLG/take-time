---
title: Create Subject
api: POST /subjects
description: "Create a new activity/subject profile."
---

<ParamField body="name" type="string" required>
  Activity name (max 40 characters)
</ParamField>

<ParamField body="type" type="string" required>
  Profile type: `study`, `training`, or `inactive`
</ParamField>

<ParamField body="color" type="string">
  Hex color code (e.g. `#6366f1`). Auto-assigned if omitted.
</ParamField>

<ParamField body="slots" type="array">
  Weekly time slots. Each slot: `{ days: number[], start: string, end: string }`
</ParamField>

<ParamField body="syllabus" type="array">
  Study topics (only for `study` type). Each: `{ title: string, duration: number }`
</ParamField>

<ParamField body="exercises" type="array">
  Exercise items (only for `training` type). Each: `{ name: string, sets: number, reps: number, weight: string }`
</ParamField>

<ParamField body="habits" type="array">
  Micro-habits (only for `inactive` type). Each: `{ name: string }`
</ParamField>

<RequestExample>
```bash cURL
curl -X POST "https://api.taketime.space/v1/subjects" \
  -H "Authorization: Bearer tt_live_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Mathematics",
    "type": "study",
    "color": "#f59e0b",
    "slots": [
      { "days": [1, 3], "start": "14:00", "end": "15:30" }
    ],
    "syllabus": [
      { "title": "Linear Algebra", "duration": 45 },
      { "title": "Calculus I", "duration": 60 }
    ]
  }'
```
</RequestExample>

<ResponseExample>
```json 201
{
  "data": {
    "id": "sub_new456",
    "name": "Mathematics",
    "type": "study",
    "color": "#f59e0b",
    "slots": [...],
    "syllabus": [...]
  },
  "error": null,
  "meta": { ... }
}
```
</ResponseExample>
