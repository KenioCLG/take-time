---
title: Update Subject
api: PATCH /subjects/{subject_id}
description: "Update an existing activity/subject. Only send the fields you want to change."
---

<ParamField path="subject_id" type="string" required>
  Subject ID to update
</ParamField>

<ParamField body="name" type="string">
  New activity name
</ParamField>

<ParamField body="color" type="string">
  New hex color
</ParamField>

<ParamField body="slots" type="array">
  Replace weekly slots (sends full array)
</ParamField>

<RequestExample>
```bash cURL
curl -X PATCH "https://api.taketime.app/v1/subjects/sub_xyz" \
  -H "Authorization: Bearer tt_live_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{ "name": "Advanced JavaScript", "color": "#818cf8" }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "data": {
    "id": "sub_xyz",
    "name": "Advanced JavaScript",
    "type": "study",
    "color": "#818cf8",
    "slots": [...],
    "syllabus": [...]
  },
  "error": null,
  "meta": { ... }
}
```
</ResponseExample>
