---
title: Delete Subject
api: DELETE /subjects/{subject_id}
description: "Permanently delete an activity and all associated blocks."
---

<ParamField path="subject_id" type="string" required>
  Subject ID to delete
</ParamField>

<RequestExample>
```bash cURL
curl -X DELETE "https://api.taketime.app/v1/subjects/sub_xyz" \
  -H "Authorization: Bearer tt_live_your_key_here"
```
</RequestExample>

<ResponseExample>
```json 200
{
  "data": {
    "deleted": true,
    "id": "sub_xyz",
    "blocks_deleted": 12
  },
  "error": null,
  "meta": { ... }
}
```
</ResponseExample>

<Warning>
  This deletes the activity **and all associated blocks**. This action is irreversible.
</Warning>
