---
title: Get Stats
api: GET /stats
description: "Get completion statistics, streaks, and heatmap data."
---

<ParamField query="period" type="string" default="week">
  Time period: `today`, `week`, `month`, or `all`
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://api.taketime.app/v1/stats?period=week" \
  -H "Authorization: Bearer tt_live_your_key_here"
```
</RequestExample>
