---
title: Update Block
api: PATCH /blocks/{block_id}
description: "Update an existing study block. Only send the fields you want to change."
---

<ParamField path="block_id" type="string" required>
  Block ID to update
</ParamField>

<ParamField body="subject_id" type="string">
  New activity/subject ID
</ParamField>

<ParamField body="start" type="string">
  New start time (`HH:MM`)
</ParamField>

<ParamField body="end" type="string">
  New end time (`HH:MM`)
</ParamField>

<ParamField body="topic" type="string">
  New topic name
</ParamField>

<ParamField body="done" type="boolean">
  Mark as completed or uncompleted
</ParamField>

<RequestExample>
```bash cURL
curl -X PATCH "https://api.taketime.space/v1/blocks/blk_abc123" \
  -H "Authorization: Bearer tt_live_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{ "done": true }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "data": {
    "id": "blk_abc123",
    "subject_id": "sub_xyz",
    "subject_name": "JavaScript",
    "date": "2026-06-11",
    "start": "09:00",
    "end": "10:00",
    "topic": "Ch. 3 - Arrays",
    "done": true
  },
  "error": null,
  "meta": { ... }
}
```
</ResponseExample>
