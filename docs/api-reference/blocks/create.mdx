---
title: Create Block
api: POST /blocks
description: "Create a new study block on the schedule."
---

<ParamField body="subject_id" type="string" required>
  Activity/subject ID to assign the block to
</ParamField>

<ParamField body="date" type="string" required>
  Date for the block (`YYYY-MM-DD`)
</ParamField>

<ParamField body="start" type="string" required>
  Start time (`HH:MM`)
</ParamField>

<ParamField body="end" type="string" required>
  End time (`HH:MM`)
</ParamField>

<ParamField body="topic" type="string">
  Optional topic name or syllabus item reference
</ParamField>

<RequestExample>
```bash cURL
curl -X POST "https://api.taketime.space/v1/blocks" \
  -H "Authorization: Bearer tt_live_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "subject_id": "sub_xyz",
    "date": "2026-06-11",
    "start": "09:00",
    "end": "10:00",
    "topic": "Arrays & Loops"
  }'
```
</RequestExample>

<ResponseExample>
```json 201
{
  "data": {
    "id": "blk_new123",
    "subject_id": "sub_xyz",
    "subject_name": "JavaScript",
    "date": "2026-06-11",
    "start": "09:00",
    "end": "10:00",
    "topic": "Arrays & Loops",
    "done": false
  },
  "error": null,
  "meta": {
    "request_id": "req_002",
    "timestamp": "2026-06-11T09:00:00Z"
  }
}
```

```json 409 Conflict
{
  "data": null,
  "error": {
    "code": "CONFLICT",
    "message": "Time slot overlaps with existing block blk_abc123 (09:00-10:00)"
  },
  "meta": { ... }
}
```
</ResponseExample>
