---
title: Get Schedule
api: GET /schedule
description: "Get the user's full weekly schedule with all configured time slots."
---

<ParamField query="week_offset" type="number" default="0">
  Week offset: `0` = current, `-1` = last week, `1` = next week
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://api.taketime.space/v1/schedule?week_offset=0" \
  -H "Authorization: Bearer tt_live_your_key_here"
```
</RequestExample>
