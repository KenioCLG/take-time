---
title: Suggest Schedule
api: POST /schedule/suggest
description: "Get AI-powered schedule suggestions based on user patterns and goals."
---

<ParamField body="goal" type="string">
  Natural language goal (e.g. "study more math this week")
</ParamField>

<ParamField body="available_hours" type="number">
  Max hours per day available for study
</ParamField>

<ParamField body="preferred_times" type="string">
  Preference: `morning`, `afternoon`, `evening`, or `any`
</ParamField>

<RequestExample>
```bash cURL
curl -X POST "https://api.taketime.app/v1/schedule/suggest" \
  -H "Authorization: Bearer tt_live_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "goal": "Focus more on math and reduce JavaScript sessions",
    "available_hours": 4,
    "preferred_times": "morning"
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "data": {
    "suggestions": [
      {
        "subject": "Mathematics",
        "recommended_slots": [
          { "days": [1, 2, 3, 4, 5], "start": "08:00", "end": "09:30" }
        ],
        "reason": "Increasing math from 2 to 5 days/week based on your goal"
      },
      {
        "subject": "JavaScript",
        "recommended_slots": [
          { "days": [1, 3, 5], "start": "10:00", "end": "10:45" }
        ],
        "reason": "Reduced from 5 to 3 days, shorter sessions"
      }
    ],
    "total_weekly_hours": 13.75,
    "apply_url": "/schedule/apply"
  },
  "error": null,
  "meta": { ... }
}
```
</ResponseExample>

<Info>
  Suggestions are not automatically applied. Review them and use `update_block` or the app UI to apply changes.
</Info>
