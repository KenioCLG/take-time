## What is MCP?

The **Model Context Protocol (MCP)** is an open standard that allows AI assistants like Claude, Cursor, and others to interact with external tools and data sources.

The Take Time MCP Server gives your AI assistant direct access to your study data — so you can manage your schedule using natural language.

| **Claude Code** | Supported | `.claude/settings.json` |

## Architecture

```
┌─────────────────┐     stdio      ┌──────────────────┐     HTTPS     ┌─────────────────┐
│   AI Assistant   │ ◄──────────► │  MCP Server      │ ◄───────────► │  Take Time API  │
│  (Claude, etc.)  │               │  (@taketimemcp)  │               │  (Supabase)     │
└─────────────────┘               └──────────────────┘               └─────────────────┘
```

The MCP Server runs as a **local process** on your machine. It communicates with the AI client via `stdio` and authenticates to the Take Time API using your API key.

<Info>
  Your data never passes through third-party servers. The MCP Server talks directly to the Take Time API.
</Info>
