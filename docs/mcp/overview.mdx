## What is MCP?

The **Model Context Protocol (MCP)** is an open standard that allows AI assistants like Claude, Cursor, and others to interact with external tools and data sources.

The Take Time MCP Server gives your AI assistant direct access to your study data вҖ” so you can manage your schedule using natural language.

The MCP Server (v1.6.0) runs as a **local process** on your machine. It communicates with the AI client via `stdio` and connects directly to your secure Take Time database using Supabase's PostgREST API. All 22 tools operate on **relational tables** (subjects, blocks, priorities, logs, profiles, subject_items) вҖ” no JSON blob indirection.

<Info>
  **Atomic & Secure:** All operations (CRUD) are executed atomically directly in the PostgreSQL relational tables using Row Level Security (RLS). Your data is isolated, and the MCP Server acts seamlessly as a bridge between the AI and your exact database rows.
</Info>
