## Available Tools

The Take Time MCP Server exposes **8 tools** to your AI assistant — 4 for reading and 4 for writing.

### Read Operations

<ResponseField name="list_blocks" type="tool">
  List study blocks with optional date range and status filters. Returns today's blocks by default.

  **Parameters:**
  | Name | Type | Required | Description |
  |------|------|----------|-------------|
  | `date` | string | No | Filter by date (`YYYY-MM-DD`). Defaults to today. |
  | `date_from` | string | No | Start of date range (`YYYY-MM-DD`) |
  | `date_to` | string | No | End of date range (`YYYY-MM-DD`) |
  | `subject_name` | string | No | Filter by activity name |
  | `done` | boolean | No | Filter by completion status |
</ResponseField>

<ResponseField name="list_subjects" type="tool">
  List all activities/subjects with their profiles, slots, and content.

  **Parameters:**
  | Name | Type | Required | Description |
  |------|------|----------|-------------|
  | `type` | string | No | `study`, `training`, or `inactive` |
</ResponseField>

<ResponseField name="get_stats" type="tool">
  Get completion statistics, streaks, heatmap data, and top subjects.

  **Parameters:**
  | Name | Type | Required | Description |
  |------|------|----------|-------------|
  | `period` | string | No | `today`, `week`, `month`, or `all` (default: `week`) |
</ResponseField>

<ResponseField name="get_schedule" type="tool">
  Get the full weekly schedule with all blocks organized by day.

  **Parameters:**
  | Name | Type | Required | Description |
  |------|------|----------|-------------|
  | `week_offset` | number | No | `0` = current week, `-1` = last week, `1` = next week |
</ResponseField>

### Write Operations

<ResponseField name="create_block" type="tool">
  Create a new study block on the schedule. Automatically checks for time conflicts.

  **Parameters:**
  | Name | Type | Required | Description |
  |------|------|----------|-------------|
  | `subject_name` | string | Yes | Name of the activity/subject |
  | `date` | string | Yes | Date (`YYYY-MM-DD`) |
  | `start` | string | Yes | Start time (`HH:MM`) |
  | `end` | string | Yes | End time (`HH:MM`) |
  | `topic` | string | No | Topic or syllabus item |
</ResponseField>

<ResponseField name="update_block" type="tool">
  Update an existing block. Use to mark as done, reschedule, or change topic.

  **Parameters:**
  | Name | Type | Required | Description |
  |------|------|----------|-------------|
  | `block_id` | string | Yes | Block ID to update |
  | `start` | string | No | New start time (`HH:MM`) |
  | `end` | string | No | New end time (`HH:MM`) |
  | `topic` | string | No | New topic |
  | `done` | boolean | No | Mark as done or undone |
</ResponseField>

<ResponseField name="delete_block" type="tool">
  Permanently delete a study block.

  **Parameters:**
  | Name | Type | Required | Description |
  |------|------|----------|-------------|
  | `block_id` | string | Yes | Block ID to delete |
</ResponseField>

<ResponseField name="create_subject" type="tool">
  Create a new activity profile (study, training, or routine).

  **Parameters:**
  | Name | Type | Required | Description |
  |------|------|----------|-------------|
  | `name` | string | Yes | Activity name (max 40 characters) |
  | `type` | string | Yes | `study`, `training`, or `inactive` |
  | `color` | string | No | Hex color (e.g. `#6366f1`). Auto-assigned if omitted. |
</ResponseField>
