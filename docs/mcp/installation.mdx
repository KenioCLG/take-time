## Prerequisites

- **Node.js 18+** installed
- A **Take Time account** with an API key
- One of the supported AI clients

## Install

```bash
npm install -g @taketime/mcp-server
```

Or use directly with `npx` (no install needed):

```bash
npx @taketime/mcp-server
```

## Configure Your Client

<Tabs>
  <Tab title="Claude Desktop">
    Edit `~/Library/Application Support/Claude/claude_desktop_config.json` (macOS) or `%APPDATA%/Claude/claude_desktop_config.json` (Windows):

    ```json
    {
      "mcpServers": {
        "taketime": {
          "command": "npx",
          "args": ["@taketime/mcp-server"],
          "env": {
            "TAKETIME_API_KEY": "tt_live_your_key_here"
          }
        }
      }
    }
    ```
  </Tab>

## Verify Installation

After configuring, restart your AI client. You should see the Take Time tools available. Try asking:

> "List my study blocks for today"

If the tools are working, you'll get a response with your block data.

## Troubleshooting

<AccordionGroup>
  <Accordion title="Tools not appearing">
    - Verify the config file path is correct for your OS
    - Check that `npx` is in your system PATH
    - Restart the AI client completely (not just the conversation)
  </Accordion>
  <Accordion title="Authentication errors">
    - Verify your API key starts with `tt_live_` or `tt_test_`
    - Check for extra spaces or quotes around the key
    - Generate a new key if the current one expired
  </Accordion>
  <Accordion title="Connection timeouts">
    - Check your internet connection
    - The Take Time API may be experiencing downtime
    - Try running `npx @taketime/mcp-server --health` to check
  </Accordion>
</AccordionGroup>
