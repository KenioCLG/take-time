## Prerequisites

- **Node.js 18+** installed
- A **Take Time account** (sign up at [taketime.space](https://taketime.space))
- One of the supported AI clients

## Install

```bash
git clone https://github.com/KenioCLG/take-time.git
cd take-time/mcp-server
npm install
```

## Authentication

The MCP Server supports two authentication methods:

<Tabs>
  <Tab title="Email/Password (Recommended)">
    Use your Take Time account credentials. The session **auto-refreshes** so the server runs indefinitely.

    ```
    TAKETIME_EMAIL=your@email.com
    TAKETIME_PASSWORD=your-password
    ```
  </Tab>
  <Tab title="Access Token">
    Use a Supabase access token from your browser session. **Expires after ~1 hour** — suitable for testing only.

    ```
    TAKETIME_API_KEY=your_access_token
    ```
  </Tab>
</Tabs>

## Configure Your Client

<Tabs>
  <Tab title="Claude Desktop">
    Edit `~/Library/Application Support/Claude/claude_desktop_config.json` (macOS) or `%APPDATA%/Claude/claude_desktop_config.json` (Windows):

    ```json
    {
      "mcpServers": {
        "taketime": {
          "command": "node",
          "args": ["/path/to/take-time/mcp-server/src/index.js"],
          "env": {
            "TAKETIME_EMAIL": "your@email.com",
            "TAKETIME_PASSWORD": "your-password"
          }
        }
      }
    }
    ```
  </Tab>

## Verify Installation

After configuring, restart your AI client. You should see the Take Time tools available. Try asking:

> "List my study subjects"

If the tools are working, you'll get a response with your activity data.

## Troubleshooting

<AccordionGroup>
  <Accordion title="Tools not appearing">
    - Verify the path to `src/index.js` is absolute and correct
    - Check that `node` is in your system PATH
    - Make sure `npm install` was run in the `mcp-server/` directory
    - Restart the AI client completely (not just the conversation)
  </Accordion>
  <Accordion title="Authentication errors">
    - Verify your email and password are correct
    - Make sure you have a Take Time account at [taketime.space](https://taketime.space)
    - If using access token, note it expires after ~1 hour
  </Accordion>
  <Accordion title="Connection timeouts">
    - Check your internet connection
    - The server needs to reach `gwujukwvufmfecqgzvrs.supabase.co`
    - Try running `node src/index.js` directly to see error messages
  </Accordion>
</AccordionGroup>
