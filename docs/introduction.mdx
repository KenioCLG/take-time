## What is Take Time?

Take Time is a **personal study planner** designed to help students organize their time using cycles, intelligent scheduling, and visual progress tracking.

With the Take Time **MCP Server**, you can connect your AI assistant (Claude, ChatGPT, Cursor, VS Code Copilot, and others) to manage your study schedule using natural language.

## Design Principles

<AccordionGroup>
  <Accordion title="Intention-based tools">
    The MCP tools are designed around **what you want to do**, not how the database works.
    `create_block` takes a subject name and time — the server handles the rest.
  </Accordion>
  <Accordion title="Offline-first app">
    The Take Time app works offline with local storage and syncs to Supabase when online.
    The MCP Server always reflects the latest synced state.
  </Accordion>
  <Accordion title="Works with any AI client">
    The MCP Server uses the standard Model Context Protocol, compatible with Claude Desktop, Claude Code, Cursor, VS Code Copilot, Windsurf, and any MCP-compatible client.
  </Accordion>
</AccordionGroup>

## Getting Started

<Steps>
  <Step title="Create an account">
    Sign up at [Take Time](https://taketime.space) and set up your first activities.
  </Step>
  <Step title="Install the MCP Server">
    Add `@taketimemcp/mcp-server` to your AI client config — runs via `npx`, no manual install needed.
  </Step>
  <Step title="Configure your AI client">
    Add your credentials to your AI client's MCP config.
  </Step>
  <Step title="Start managing with AI">
    Ask your assistant: "What do I have scheduled for today?"
  </Step>
</Steps>
