## Heatmap

The heatmap is a visual calendar that lights up based on your daily productivity. Each cell represents a day, and the color intensity reflects how many tasks you completed.

The heatmap counts:
- Completed study blocks
- Finished exercises
- Checked micro-habits

## Metrics Available

| Metric | Description |
|--------|-------------|
| `total_blocks` | Total blocks in the period |
| `completed_blocks` | Blocks marked as done |
| `completion_rate` | Ratio of done/total (0-1) |
| `streak_days` | Consecutive days with at least one completion |
| `total_minutes` | Total study time in minutes |
| `heatmap` | Daily completion counts (`{ "YYYY-MM-DD": count }`) |

## Periods

The `get_stats` endpoint accepts these periods:

| Period | Description |
|--------|-------------|
| `today` | Current day only |
| `week` | Current week (Mon-Sun) |
| `month` | Current calendar month |
| `all` | All time |
