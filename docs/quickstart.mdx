## 1. Create your account

Sign up at [taketime.space](https://taketime.space) with email and password. Add a few activities and study blocks to have data to work with.

## 2. Install the MCP Server

```bash
git clone https://github.com/KenioCLG/take-time.git
cd take-time/mcp-server
npm install
```

## 3. Configure your AI client

Add the MCP Server to your AI client using your Take Time credentials:

<Tabs>
  <Tab title="Claude Code">
    Edit `~/.claude/settings.json`:

    ```json
    {
      "mcpServers": {
        "taketime": {
          "command": "node",
          "args": ["/path/to/take-time/mcp-server/src/index.js"],
          "env": {
            "TAKETIME_EMAIL": "your@email.com",
            "TAKETIME_PASSWORD": "your-password"
          }
        }
      }
    }
    ```
  </Tab>
  <Tab title="Claude Desktop">
    Edit `~/Library/Application Support/Claude/claude_desktop_config.json` (macOS) or `%APPDATA%/Claude/claude_desktop_config.json` (Windows):

    ```json
    {
      "mcpServers": {
        "taketime": {
          "command": "node",
          "args": ["/path/to/take-time/mcp-server/src/index.js"],
          "env": {
            "TAKETIME_EMAIL": "your@email.com",
            "TAKETIME_PASSWORD": "your-password"
          }
        }
      }
    }
    ```
  </Tab>
  <Tab title="Cursor">
    Edit `.cursor/mcp.json` in your project root:

    ```json
    {
      "mcpServers": {
        "taketime": {
          "command": "node",
          "args": ["/path/to/take-time/mcp-server/src/index.js"],
          "env": {
            "TAKETIME_EMAIL": "your@email.com",
            "TAKETIME_PASSWORD": "your-password"
          }
        }
      }
    }
    ```
  </Tab>
</Tabs>

<Note>
  Replace `/path/to/take-time` with the actual absolute path where you cloned the repository.
</Note>

## 4. Try it out

Restart your AI client and try these prompts:

> "What do I have scheduled for today?"

> "Show me my weekly stats"

> "Create a Math study block tomorrow from 9am to 10:30am about Linear Algebra"

If the tools respond with your data, you're all set!
